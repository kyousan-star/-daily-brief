yAyusi

Amazon affiliate + Amazon Finds 的功能演示型打法可以拆；在当前可核验证据里，yAyusi 的链路完整度和可借鉴价值最高。

打法可拆样本
主渠道

YouTube 有 4 条高相关内容，最高约 16K views；TikTok 手动截图有索链评论，但热度只算中等。

一句话 takeaway

优先学“痛点 hook + 功能演示 + Amazon 承接”，同时继续验证热度是否足够。

值得学
  • 把 auto tracking 拍成“一个人拍摄”的解决方案，而不是只讲参数。
  • 标题和 caption 直接铺核心关键词：yAyusi、ES-B180、face tracking、tripod。
  • 用 Amazon Finds / affiliate 链路做低成本种草承接。
风险边界
  • 视频偏旧且热度不够强，不能作为成功投放样本。
  • 多条 YouTube 内容可能是同一 affiliate 长尾分发，不代表品牌官方主动投放。

Yeaki

UGC/Amazon Finds + paid/affiliate link + Amazon Shopping 的表达可拆，但互动表现偏弱。

弱互动打法样本
主渠道

YouTube 有高相关内容；TikTok 手动截图仅 2 likes / 0 comments / 1 save / 1 share，不能算热度样本。

一句话 takeaway

可学 creator kit 表达和用户问题，但不能学它的投放强度。

值得学
  • 把 tripod 包装成 content creator 工具，而不是单纯手机支架。
  • UGC 文案覆盖 360 face tracking、可充电遥控、手机兼容、ring light 兼容。
  • 用 same-day delivery / @Amazon / Amazon Finds 强化购买便利。
风险边界
  • 当前 TikTok 互动弱，不能证明 Yeaki 站外做得好。
  • Amazon Shopping 内容不能算品牌社媒创意。
  • 部分 YouTube 内容像自动化产品概览，需区分真实达人和自动生成内容。
